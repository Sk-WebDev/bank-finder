# bank-finder
Bank Details Finder using IFSC Code

This web application allows users to retrieve bank details using an IFSC code.
It is built using Node.js, Express, and EJS, and consumes a publicly available IFSC API to fetch accurate bank information.

The application provides a simple and user-friendly interface where users can enter an IFSC code and instantly view the corresponding bank details.

Features

Fetch bank details using IFSC code

Uses a publicly available IFSC API

Server-side rendering with EJS

Simple and beginner-friendly project structure



Tech Stack

Node.js
Express.js
EJS
Axios
HTML, CSS

Installation & Setup

Clone the repository

git clone <repository-url>
cd <project-folder>


Install dependencies

npm install
or
npm i


Run the application

nodemon index.js


Open in browser

http://localhost:3000
